# Sales Data Analysis

This project analyzes sales transaction data using Python (Pandas) and SQL-style queries.

## Objectives
- Calculate total order value
- Analyze revenue by product and category
- Demonstrate basic data analysis workflow

## Tools Used
- Python
- Pandas
- SQL (conceptual)
- CSV data

## Key Insights
- Electronics generate the highest revenue
- Laptop is the top-selling product by revenue
- Stationery has low revenue despite high quantity

## Files
- sales_analysis.py – Python analysis script
- sales_analysis.sql – SQL queries for analysis
- sales_data.csv – Sample sales dataset
